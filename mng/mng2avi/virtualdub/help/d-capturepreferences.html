Preferences

Controls basic video capture settings.

Default capture driver
Selects which capture driver VirtualDub will select when entering capture mode. If the specified driver cannot be found, the first available one is used.
Default capture file
Sets the default capture filename used by VirtualDub when entering capture mode.
Save current capture settings, audio format, video format, video compression settings as default
Saves a group of settings as the default on startup. Due to a limitation in Video for Windows, video compression settings cannot be saved for compatibility mode.
Per-driver settings
Select a startup display mode for video capture, and possibly disable display when menus or dialogs are displayed which overlap the video display. This may help if you experience performance problems with one or more of the video display modes.
